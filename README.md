# Project_PetGPT

반려동물 생애주기 관리 웹 서비스. 사용자 위치를 기반으로 주변의 적절한
가게·시설·정보를 연결해주는 플랫폼.

## 페이지별 요구사항

### 1. 입양 (가족 찾기)
- [ ] 사용자 라이프스타일 입력 (주거, 가족, 활동량, 알러지)
- [ ] 조건 기반 품종·성향 추천
- [ ] 위치 기반 보호소·입양센터 안내
- [ ] 동물보호관리시스템 공공데이터 연동
- [ ] 품종별 특성 정보 제공 (수명, 양육비, 주의 질환)
- [ ] 입양 전 체크리스트

### 2. 식단 (맞춤 영양)
- [ ] 반려동물 프로필 등록
- [ ] RER·MER 기반 하루 권장 칼로리 계산
- [ ] 건강 고민별 추천 영양 성분 안내
- [ ] 시중 사료 비교 검색
- [ ] 반려동물 위험 음식 목록
- [ ] 다묘·다견 프로필 동시 관리
- [ ] 체중 변화 그래프

### 3. 용품점 찾기
- [ ] 시/도 → 시군구 → 동 단위 위치 입력
- [ ] 지도와 목록으로 매장 표시
- [ ] 영업시간·취급 품목 정보
- [ ] 서비스 필터링 (미용, 호텔 등)
- [ ] 전화 걸기·길찾기 외부 연결
- [ ] 사용자 평점·리뷰

### 4. 건강 수첩
- [ ] 케어 일정 등록 (예방접종, 구충 등)
- [ ] D-day 기반 다가오는 일정 표시
- [ ] 반복 주기 자동 갱신
- [ ] 진료 기록 입력 (진단, 처방, 비용)
- [ ] 진료 시 사진 첨부
- [ ] 위치 기반 동물병원 찾기
- [ ] 체중 변화 그래프
- [ ] 진료 기록 CSV·PDF 내보내기

### 5. 마지막 안녕
- [ ] 위치 기반 장례식장 안내
- [ ] 시설 유형·서비스 정보 (개별 화장, 봉안당 등)
- [ ] 정식 허가 여부 표시
- [ ] 비대면 상담 신청
- [ ] 장례 절차 안내 콘텐츠

### 6. 메인 (홈)
- [x] 5개 서비스 진입점
- [ ] 등록된 반려동물 요약
- [ ] 다가오는 일정 알림
- [ ] 위치 정보 페이지 간 공유
- [ ] 실종 신고 기능



## 공통 요구사항
- [ ] 데이터 영속성 (SQLite 또는 DB)
- [ ] 사용자 계정·로그인
- [ ] 매장·시설 데이터 출처 관리
- [ ] 근처 동물 병원, 공원 등 시설에 대한 평가 시스템
- [ ] 위치 권환 요청 및 처리
- [ ] 주변 인프라 카테고리 분류
- [ ] 주변 사용자 표시
